3. Custom SMT transforms data (removes null bytes, routes topics)
4. Avro Converter serializes data using Schema Registry
5. CDC events published to Kafka topics (format: database_tablename)
6. Kafka Brokers store events with replication
7. JMX Exporters expose metrics from all components
8. Prometheus scrapes and stores metrics
9. Alert rules evaluated every 10 seconds
10. Alertmanager processes and routes alerts
11. Calert formats and sends alerts to Google Chat
12. Grafana provides visualization dashboards
13. Downstream consumers read CDC events from Kafka topics
